# 🎙️ VocalApp

Android-приложение для анализа голоса в реальном времени. Разработано как инструмент обратной связи при самостоятельном изучении вокальной техники.

## Возможности

- **Спектральный анализ в реальном времени** — FFT-анализ входящего аудио с микрофона на основе библиотеки JTransforms
- **Визуализация формант** — выделение формантных пиков, позволяющее отслеживать тембровые характеристики голоса
- **Piano roll / нотный дисплей** — отображение текущей высоты тона в привязке к нотному ряду
- **Живая обратная связь** — мгновенное отображение изменений при пении или речи

## Стек

| Компонент | Технология |
|---|---|
| Язык | Kotlin |
| UI | Jetpack Compose |
| FFT | JTransforms |
| Сборка | Gradle (Kotlin DSL) |
| Минимальный SDK | Android 8.0+ |

## Структура проекта


## Сборка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/doctorshtopor/VocalApp.git
   ```
2. Открой проект в **Android Studio**
3. Запусти на устройстве или эмуляторе через `Run > Run 'app'`

> Для корректной работы приложению требуется разрешение на использование микрофона.

## Мотивация

Приложение создавалось как персональный инструмент при изучении классической вокальной техники (bel canto). Спектральный анализ помогает отслеживать формантную настройку, контролировать певческий форнант и сравнивать свой тембр с эталонными образцами — то, что обычно недоступно без профессионального оборудования.
