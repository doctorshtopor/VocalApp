# 🎙️ VocalApp

Android-приложение для анализа голоса в реальном времени. Разработано как инструмент обратной связи при самостоятельном изучении вокальной техники. 

Приложение создавалось для личного использования, поэтому в нем есть баги, которые автору не мешают. Спектральный анализ помогает отслеживать формантную настройку и сравнивать свой тембр с эталонными образцами - это помогает лучше выявлять недостатки своего исполнения и выделять аспекты вокала над которыми стоит работать.

## Разработка с помощью AI
 
Так как автор не является андроид разработчиком и приложение изначально не задумывалось как коммерческий продукт, то код полностью написан различными нейросетями, однако сборка приложения выглядит ровно так, как хотел автор (если не учитывать баги). 


## Возможности

- **Спектральный анализ в реальном времени и из файла** — FFT-анализ входящего аудио с микрофона или файла на основе библиотеки JTransforms
- **Пиковый детектор** - по долгому нажатию на экран приложение начинает записывать пиковые значения спектра во времени
- **Piano roll** — отображение текущей высоты тона в привязке к нотному ряду
- **Живая обратная связь** — мгновенное отображение изменений при пении или речи

## Стек

| Компонент | Технология |
|---|---|
| Язык | Kotlin |
| UI | Jetpack Compose |
| FFT | JTransforms |
| Сборка | Gradle (Kotlin DSL) |
| Минимальный SDK | Android 8.0+ |

## Структура проекта

```
VocalApp/
├── app/
│   └── src/main/        # Основной исходный код
├── build.gradle.kts     # Корневой конфиг сборки
├── settings.gradle.kts  # Настройки проекта и зависимостей

```

## Сборка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/doctorshtopor/VocalApp.git
   ```
2. Откройте проект в **Android Studio**
3. Запустите на устройстве или эмуляторе через `Run > Run 'app'`

> Для корректной работы приложению требуется разрешение на использование микрофона.
